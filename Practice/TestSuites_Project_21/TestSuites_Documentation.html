Test Suites Python testcase Project

In this project you will get complete idea of how to create proper Test suite in Python Selenium and also how to create python packages

  • Creating test cases and test suites in Python Unittest Framework
  • Executing test Suites

In This we are using TestLoader in our TestSuite package To run all test suites properly

The TestLoader class is used to create test suites from classes and modules. Normally, there is no need to create an instance of this class; the unittest module provides an instance that can be shared as unittest.
